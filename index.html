What is Orna?

Orna is a good tool, designed for atomize Web UI.

Atomic CSS - is a model or style of coding where each of properties has own style class.

Atomic CSS is good for fast creating the readable and easy changeable User Interface.


What Orna do?

If you plug orna.js into html document you will have a new skill that give you write css properties with values in class attribute and it will means class and property in one and call it atom. In orna.css you find classes with few properties and that calls molecules. Molecules make your work easy when you the properties have many value. Molecules give it like pack. Also you have other functions that help you in your creation.


How Orna works?

Just write css properties in class attribute then write underscore and value Use orna.css classes (molecules) if you need. Event in orna create with jQuery and onclick, onmouseover and other event attribute.


Examples

Write css properties like classes
class="background-color_black width_50% height_100px"
